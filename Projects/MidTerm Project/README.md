Through machine learning techniques I analyzed data to create predictions while learning the complete machine learning project process for my midterm project. My research process started with data collection followed 
by data preprocessing which included cleaning pending and incorrect data then I used visualizations to analyze data patterns and tested multiple machine learning models to find the best performing model. The model 
evaluation results showed how well both regression and classification models performed through feature engineering and model evaluation metrics. The results showed that some models generalized better than others 
because more complex data required different tuning approaches to increase prediction accuracy. The results demonstrated that effective data preparation and appropriate model selection with evaluation methods play a 
crucial role in developing trustworthy AI systems which can be used in actual business operations.
