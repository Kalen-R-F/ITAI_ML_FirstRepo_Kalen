The research project I conducted at the laboratory involved using unsupervised learning methods to discover hidden patterns which existed in data that lacked any labels. I used K-Means clustering to group iris 
flowers into clusters based on their features and applied Principal Component Analysis (PCA) to reduce the dataset from four dimensions down to two for visualization. I evaluated clustering performance using the 
elbow method to choose a reasonable number of clusters and then visualized the reduced data with PCA. The results showed that the first two principal components captured over 95% of the dataset’s variance, meaning 
most of the important information could be represented in just two dimensions. The study showed how unsupervised learning methods can extract data patterns from datasets while dimensionality reduction techniques 
enable easier visual representation and interpretation of complex datasets.
