My research objective focused on determining whether the machine learning model produced equitable results when tested with various demographic groups, particularly males and females from the Adult Income dataset. I 
trained a classification model and evaluated its performance through subgroup analysis which used accuracy, False Positive Rate (FPR) and False Negative Rate (FNR) as evaluation metrics. I checked the model error 
rates for different user groups to find evidence of bias and discriminatory effects. The results showed that the model did not perform equally across groups, with differences in error rates revealing possible unfair 
treatment. The study showed that accurate machine learning models still contain bias, which demonstrates the need for fairness testing together with ethical evaluation and bias reduction methods when organizations 
implement AI technology for decision-making processes in hiring and loan approval.
