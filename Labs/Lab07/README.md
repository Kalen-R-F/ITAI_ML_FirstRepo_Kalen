The Titanic passenger survival prediction problem which I wanted to solve required me to predict which passengers survived because it could be solved through binary classification. I started the process by preparing 
the dataset through various steps which included transforming all categorical data into numerical formats and building the dataset for further analysis. I then trained a classification model, such as logistic 
regression or a decision tree, to predict survival based on features like age, sex, and passenger class. The results showed that the model could achieve a decent accuracy which confirmed that certain factors 
especially gender and class strongly influenced survival outcomes. The displayed classification models demonstrated their ability to discover data patterns while producing predictions for categorical results through 
understanding their learned relationships.
